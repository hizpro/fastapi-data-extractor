# 🚀 Mesin Ekstraksi Data (Versi Enterprise)

Proyek ini adalah hasil eksperimen untuk membangun API pengeruk data yang tangguh dan cepat menggunakan Python.

## Mengapa Proyek Ini?
Saya membangun ini sekedar ingin berbagi, ini terstuktur dan dilengkapi midleware pengukur performa.

## Fitur Utama
* **Cepat:** Menggunakan sistem asinkron agar tidak antre.
* **Transparan:** Memberitahu pengguna berapa lama waktu yang dibutuhkan untuk mengambil data.
* **Tahan Banting:** Tidak mudah eror meski website target berantakan.
